# Cloned Repositories

This directory contains code implementations of attention-based models for sequential user behavior.

## Repo 1: RecGRELA
- URL: https://github.com/TBI805/RecGRELA
- Purpose: Gated Rotary-Enhanced Linear Attention with Rank Modulation for Long-term Sequential Recommendation.
- Location: code/rec_grela/
- Key files: `run_RecGRELA.py`, `RecGRELA.py`.
- Notes: Requires RecBole framework.

## Repo 2: PANTHER
- URL: https://github.com/WeChatPay-Pretraining/PANTHER
- Purpose: Official repository for "PANTHER: Generative Pretraining Beyond Language for Sequential User Behavior Modeling" (2025).
- Location: code/panther/
- Key files: `train_behseq.py`, `cct_preprocess.py`.
- Notes: Focuses on credit card transaction datasets.

## Repo 3: FuXi-Linear
- URL: https://github.com/USTC-StarTeam/fuxi-linear
- Purpose: FuXi-Linear: Unleashing the Power of Linear Attention in Long-term Time-aware Sequential Recommendation.
- Location: code/fuxi_linear/
- Key files: `main.py`.
- Notes: Focuses on time-aware recommendation.
